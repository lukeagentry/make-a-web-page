The Begining of HTML Programing

HTML & The Internet

HTML stands for Hypertext Markup Language. The web contains billions of HTML documents. These documents are all stored on servers. When you "Google" something, your computer submits a HTTP request to a server. The server finds the document you are looking for and sends that document to your computers browser. The browser interprets the page and displays it on your computer.

All HTML documents are made up of boxes. These boxes can contain images, content and even other shapes. Front-end web developers look at all websites as if they are laid out in boxes. It can be useful to take something as simple as a pencil to scetch out these boxes. To make these boxes, a "div" tag is used at the begining and end of each box.

Tags and Elements

Tags and elements make up every page of HTML code. This is how computers read the documents. Computers are stupid in that they can't catch basic coding mistakes that a human can catch. They simply do what they are told to do by the coding within the HTML document. Therefore, having a period out of place or forgetting to close a tag can cause major problems.

Inline vs Block Elements

Elements of an HTML document are either Inline or Block. The only major difference is that block elements form an "invisible box" around the content inside of them.

Google Chrome

Google Chrome is a unique internet browser. Whenever you open a internet page using Chrome, a menu appears in the upper right hand corner. You can click on this menu and go to "tools". From there you can go to "developer tools". When you open developer tools on a webpage, you can see all of the HTML coding for that page. This is a great tool when you are first starting to code. By looking at existing code and changing the code, you can start to figure out how things are programed.

CSS

CSS stands for cascading style sheets. These style sheets are what is used to determing how a page looks on the internet. You can control virtualy every aspect of a page if you know the correct CSS to use. This page can be used as a resource for looking up different CSS. It would be almost impossible to learn every CSS variable, so it is common to use a website like this.
